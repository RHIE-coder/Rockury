# Java Profile (Spring Boot + Gradle)

## Overview
Java 기반 Spring Boot 프로젝트를 위한 프로파일

---

## Stack (Recommended)
- **Framework**: Spring Boot
- **Build**: Gradle
- **Language**: Java 17+
- **Test**: JUnit 5, Mockito
- **Coverage**: JaCoCo
- **Style**: Checkstyle or Spotless

---

## Project Characteristics
- 모듈 경계는 `api` / `service` / `domain` / `infra` 기준 권장
- 설정은 `application.yml` 중심

---

## Commands (Placeholder)
- `./gradlew test`
- `./gradlew check`
- `./gradlew build`

---

## Notes
- 게이트 명령은 플러그인/프로파일 규칙에서 정의
