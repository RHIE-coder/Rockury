## How-To Guides

### 환경 설정
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run start

# 빌드
npm run make
```

### 새 기능 추가
1. Feature 폴더 생성: `src/renderer/features/[feature-name]/`
2. 컴포넌트 작성
3. Public API 정의 (index.ts)
4. 테스트 작성

### 디버깅
- **Renderer 디버깅**: DevTools (Ctrl+Shift+I)
- **Main 디버깅**: VS Code 디버거 설정
