## Full Learning Pack

```

- 이름: [프로젝트명]
- 경로: [프로젝트 경로]

## 분석 대상
- package.json
- 설정 파일 (tsconfig, vite.config 등)
- 주요 소스 코드

1. 기술 스택 분석 및 학습 우선순위
2. 각 핵심 기술별 학습 가이드
3. 온보딩 문서
4. 학습 로드맵

## 대상 독자
- [ ] 완전 초보자
- [ ] 주니어 개발자
- [ ] 시니어 개발자 (새 기술 학습)

## 출력 위치
- docs/learning/
```

## Learning Material Standards

### 콘텐츠 원칙
1. **실용성**: 이론보다 실제 적용 중심
2. **맥락**: 프로젝트 코드와 연결
3. **점진성**: 쉬운 것에서 어려운 것으로
4. **명확성**: 전문용어는 설명과 함께

### 코드 예시 규칙
- 실제 프로젝트 코드 우선
- 주석으로 핵심 포인트 표시
- 실행 가능한 코드 제공
- Before/After 비교 포함

### 참고 자료 선정 기준
- 공식 문서 우선
- 최신 버전에 맞는 자료
- 한국어 자료 있으면 함께 제공
- 영상/블로그는 검증된 것만
